# 🐻 Build-AI-Bare

> *"Stuff it with whatever you need."*

**Build-AI-Bare** is a free, open source, no-code AI configurator. Pick a purpose, a personality, some tools, give it a name — and walk out with a ready-to-use system prompt in 60 seconds. No API keys. No coding. No account required.

It works with Claude, ChatGPT, or any AI platform that accepts a system prompt.

---

## ✨ What it does

Build-AI-Bare lets anyone — technical or not — custom-build their own AI assistant by making simple choices:

- 🎯 **Purpose** — Personal assistant, creative writing, business tool, research, coding, customer support
- 🧠 **Personality** — Professional, warm, witty, socratic, concise, or mentor
- 🔧 **Tools** — Web search, code execution, file reading, email & calendar, image generation, memory
- 🏷️ **Identity** — Give your AI a name and describe who will use it
- ⚙️ **Guardrails** — Set response length, creativity level, and topics to avoid

The result is a **system prompt** — a plain-English instruction set you can paste into any AI platform to bring your custom AI to life.

## 💾 How to use your system prompt

Once you've built your AI, copy the generated system prompt and paste it into:

- **Claude** → Start a new chat → paste at the top
- **ChatGPT** → Settings → Customize ChatGPT → paste into instructions
- **Claude Projects** → Create a project → add as project instructions
- **Any OpenAI-compatible API** → use as the `system` message

## 🤝 How to contribute

Never contributed to open source before? That's fine — neither had most people the first time.

Here's the simplest way to help:

**1. Fork the repo**
Click the **Fork** button at the top right of this page. This makes your own personal copy of the project.

**2. Make your changes**
Edit `index.html` directly on GitHub, or download it, edit it locally, and re-upload.

**3. Open a Pull Request**
Click **Contribute → Open Pull Request** on your fork. Describe what you changed and why. We'll review it together.

**Ideas for contributions:**
- New purpose categories (e.g. healthcare, education, legal)
- New personality types
- More tool options
- Translations into other languages
- Accessibility improvements
- Dark/light mode toggle
- Mobile layout improvements
- A gallery of pre-built community configurations

## 🐾 Origin story

Build-AI-Bare was born from a simple observation:

> *"I'm surprised they don't have a 'Build-A-Bear' option for AI yet."*
> — A roommate, being accidentally prophetic

The idea: building a custom AI should feel as easy and fun as building a stuffed animal. Pick the parts you want. Stuff it. Walk out with something that's yours.

The first AI ever built with Build-AI-Bare was named **Zabba Doo**. 🐾
